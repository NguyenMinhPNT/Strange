## 1. PROJECT OVERVIEW & DECISIONS LOG

### App Identity

- **Name:** Strange
- **Purpose:** Time management app — track focused work sessions across Learning, Projects, and Habits using Pomodoro or Deep Work timers
- **Target:** Single user, fully offline, mobile-first (iOS + Android)
- **Design Philosophy:** Claymorphism — bold, puffy, joyful UI with Swiss flag red dominance

### Confirmed Decisions (from Product Owner)

| # | Decision | Choice | Impact |
|---|----------|--------|--------|
| D1 | Tab Categories | 3 fixed tabs: **Learning** / **Projects** / **Habits** | Single `CardType` enum, shared card list |
| D2 | Timer Types | 2 modes: **Pomodoro** (countdown) + **Deep Work** (stopwatch) | Separate Cubit states and UIs |
| D3 | Card Interaction | Long-press + drag = reorder; Swipe end→start = delete; Swipe start→end = archive | `ReorderableListView` + custom `Dismissible` |
| D4 | Card Customization | Name + color picker (custom palette + free hex) | `colorHex` stored as String in DB |
| D5 | Data Persistence | All sessions saved locally in Drift for Stats insight | `sessions` table with timer metadata |
| D6 | DI | get_it + injectable (code-gen service locator) | Type-safe, compile-time verified DI |
| D6b | Dialogs | Flutter built-ins (showDialog, showModalBottomSheet, ScaffoldMessenger) | No extra package; full theme support |
| D6c | i18n | flutter_localizations + intl (ARB-based) | Official Flutter i18n pipeline |
| D7 | Stats Charts | Heatmap (circular, GitHub-style) + Column chart + Pie chart | `fl_chart` + CustomPainter |
| D8 | Deep Work Pause | Configurable max pause duration — session auto-ends if exceeded | Timer monitors pause elapsed in settings |
| D9 | Notifications | Out of MVP scope (Sprint 8+) | No `flutter_local_notifications` in v1 |
| D10 | Background Timer | Continues when app minimized | Requires `flutter_foreground_task` |

### 4.1 Color Palette

```
PRIMARY SURFACE COLORS (30% White family)
  Surface Base:        #FFFFFF   — main background
  Surface Elevated:    #F8F8F8   — slightly off-white card base
  Surface Muted:       #F2F2F2   — subtle section dividers

PRIMARY BRAND (50% Swiss Red family)
  Primary:             #D52B1E   — Swiss flag red (main CTAs, active tabs, FAB)
  Primary Dark:        #B01F14   — pressed / active states
  Primary Light:       #E85A50   — lighter red for highlights
  Primary Container:   #FDDBD9   — soft red chip / badge background
  On Primary:          #FFFFFF   — text/icon on red surface

SEMANTIC & ACCENT (20% Others)
  Gold Accent:         #F5A623   — Deep Work indicator, warm accent
  Success:             #34C759   — session completed indicator
  Warning:             #FF9500   — pause warning (max pause approaching)
  Error:               #FF3B30   — delete confirmation
  Text Primary:        #1C1C1E   — main text (iOS dark)
  Text Secondary:      #6C6C70   — subtitles, hints
  Text On Surface:     #3A3A3C   — body text on white surface

CLAYMORPHISM SHADOW SYSTEM
  Clay Shadow Primary: rgba(213, 43, 30, 0.25)   — colored red drop shadow
  Clay Shadow Neutral: rgba(0, 0, 0, 0.10)        — subtle base shadow
  Clay Highlight:      rgba(255, 255, 255, 0.80)  — top-left inner light

DARK MODE OVERRIDES
  Surface Base:        #1C1C1E
  Surface Elevated:    #2C2C2E
  Text Primary:        #FFFFFF
  Clay Shadow Primary: rgba(213, 43, 30, 0.40)    — deeper colored shadow
```

### 4.2 Claymorphism Style Principles

```
BORDER RADIUS
  Card (clay_card):      28–32dp
  Button (clay_button):  20–24dp
  Container:             16–20dp
  FAB:                   20dp (if custom) or CircularFAB

SHADOW (BoxShadow)
  Outer shadow 1 (drop):  offset(4, 8), blur 16, spread 0, color: Clay Shadow Primary
  Outer shadow 2 (base):  offset(2, 4), blur  8, spread 0, color: Clay Shadow Neutral
  — Pressed state: shadows shrink to (2,4 / 1,2) — "clay pressed" effect

CARD SURFACE
  Color: Surface Elevated (#F8F8F8 light / #2C2C2E dark)
  Border: none (shadows do the work)
  Padding: 16–20dp

TYPOGRAPHY
  App Title "Strange":   Font: Pacifico or Lobster (script, "fly-bướm")
                         Size: 28sp, color: Primary #D52B1E
  Card Name:             Font: Inter SemiBold, 16sp
  Timer Display:         Font: SF Mono / Roboto Mono, 56sp (large monospace)
  Body:                  Font: Inter Regular, 14sp
  Caption:               Font: Inter Regular, 12sp, color: Text Secondary
```

```

PomodoroRunning  → [Pause]        → PomodoroPaused
PomodoroRunning  → [phase ends]   → PomodoroPhaseComplete (auto-transition after 1.5s)
PomodoroRunning  → [End Session]  → PomodoroEnded(completedRounds)
PomodoroPaused   → [Resume]       → PomodoroRunning
PomodoroPaused   → [End Session]  → PomodoroEnded(completedRounds)
PomodoroPhaseComplete → [auto/skip] → PomodoroRunning(next phase)
PomodoroEnded    — saves session → pop with result
```

**Pomodoro phase cycling:**
```
Round 1: Work → Short Break
Round 2: Work → Short Break
...
Round N (= interval): Work → Long Break
Repeat cycle. User can "Skip Break" to go back to Work immediately.
```

**Deep Work Timer States:**
```
DeepWorkInitial  → [Start]   → DeepWorkRunning(elapsed: 0)
DeepWorkRunning  → [Pause]   → DeepWorkPaused(pauseElapsed: 0)
DeepWorkRunning  → [End]     → DeepWorkEnded(totalWorkSec)

DeepWorkPaused   → [End]     → DeepWorkEnded(totalWorkSec)
DeepWorkEnded / DeepWorkAutoEnded — saves session → pop with result
```

**Background timer mechanism:**
- On Start: write `PREF_TIMER_START_EPOCH` + `PREF_TIMER_STATUS` + card/type info to SharedPrefs

- `flutter_foreground_task` posts persistent notification:
  - Pomodoro: "🍅 Work — 18:42 remaining · Round 2/4"

- On app resume: `TimerService` reads prefs → reconstructs elapsed = now - startEpoch + savedElapsed
- On finish/end: clear all `PREF_TIMER_*` keys

## 7. TIMER ENGINE

### 7.1 TimerService Architecture

```
TimerService (in working/data/datasources/)
├── _ticker: Stream<int>            — 1-second tick via periodic Stream
├── _foregroundTask: FlutterForegroundTask
├── start(config: TimerConfig)
├── pause()
├── resume()
├── stop()
└── _persistState()                 — writes to SharedPreferences on every state change
```

**TimerConfig (union type):**
```dart
sealed class TimerConfig {
  factory TimerConfig.pomodoro({
    required int cardId,
    required PomodoroSettings settings,
    int resumeRound,
    PomodoroPhase resumePhase,
    int resumeElapsedSec,
  }) = _PomodoroConfig;

  factory TimerConfig.deepWork({
    required int cardId,
    required DeepWorkSettings settings,
    int resumeElapsedWorkSec,
    int resumeTotalPauseSec,
  }) = _DeepWorkConfig;
}
```

### 7.2 Session Persistence Flow

**On session complete/end:**
```
1. Cubit receives EndSession event
2. TimerService.stop() → clears SharedPrefs
3. EndSessionUseCase(sessionData) called
4. SessionRepositoryImpl.saveSession(session) → Drift INSERT
5. GoRouter.pop() with SessionResult
6. HomeState reloads (no change needed — sessions are for stats only)
```

**Crash recovery on app open:**
```
1. app.dart → DI setup → check PREF_TIMER_STATUS
2. If 'running' or 'paused' → reconstruct TimerConfig from prefs
3. Navigate to WorkingPage with recovery dialog:
   `showDialog()` "Resume your [Card Name] session?" → Yes / No
4. If Yes: start Cubit with recovered config (resume mode)
5. If No: clear timer prefs, stay on home
```

### 7.3 Pomodoro Round Logic

```dart
// On each phase end (elapsedSeconds >= targetSeconds):
void _onPhaseComplete() {
  if (currentPhase == PomodoroPhase.work) {
    completedRounds++;
    if (completedRounds % shortBreakInterval == 0) {
      nextPhase = PomodoroPhase.longBreak;
    } else {
      nextPhase = PomodoroPhase.shortBreak;
    }
    emit(PomodoroPhaseComplete(completedRounds, nextPhase));
    // Auto-transition after 1500ms delay (with celebration animation)
  } else {
    // Break ended → next work phase
    nextPhase = PomodoroPhase.work;
    emit(PomodoroPhaseComplete(completedRounds, nextPhase));
  }
}
```

### 7.4 Deep Work Auto-End Logic

```dart
// On each tick while paused:
void _onPauseTick() {
  pauseElapsedSeconds++;
  totalPauseSeconds++;
  if (pauseElapsedSeconds >= maxPauseSeconds) {
    _autoEndSession(reason: 'max_pause_exceeded');
  } else {
    emit(DeepWorkPaused(
      elapsedWork: elapsedWorkSeconds,
      pauseElapsed: pauseElapsedSeconds,
      maxPause: maxPauseSeconds,
    ));
  }
}

// PauseBarIndicator shows remaining pause time:
// remaining = maxPauseSeconds - pauseElapsedSeconds
// bar fills from right as pause grows → urgent visual warning
```

## 11. KEY TECHNICAL DECISIONS & RISKS

### Decision Rationale

| Decision | Alternative Considered | Why Chosen |
|----------|------------------------|------------|
| get_it + injectable for DI | GetX DI (Get.put/find) | Type-safe compile-time verified; works without BuildContext; code-gen via injectable reduces boilerplate |
| Flutter built-ins for dialogs | GetX dialogs (Get.dialog) | No extra dependency; dialogs inherit app theme automatically |
| flutter_localizations for i18n | GetX translations | Official Flutter ARB pipeline; tooling support (Flutter IDE plugins, `flutter gen-l10n`) |
| GoRouter for navigation | GetX navigation (Get.to/off) | GoRouter is more robust for deep links, typed routes, and nested navigation; no conflict with Navigator 2.0 |
| Drift over Hive | Hive (NoSQL), SQLite (raw) | Type-safe SQL needed for stats aggregations (date ranges, GROUP BY card_type, SUM durations) |
| `ReorderableListView` (SDK) | `flutter_drag_and_drop_lists` package | Built-in widget, no extra dependency, integrates cleanly with `Dismissible` |
| CustomPainter for heatmap | `heatmap_calendar` package | Full control over circular dot shape, 5-level red intensity, tap interactions, layout |
| `flutter_foreground_task` | `workmanager` | Provides both Foreground Service (Android) + background task (iOS); simpler timer-specific API |
| Freezed for entities | Manual Equatable classes | `copyWith`, pattern matching, and equality in one codegen step |
| Feature-First folder structure | Layer-first (all blocs together) | Each feature is independently testable and removable; scales better as features grow |

### Risks & Mitigations

| Risk | Likelihood | Mitigation |
|------|------------|------------|
| get_it injectable code-gen out of sync | Low | Run `build_runner watch` during development; always commit the generated `.config.dart` file |
| iOS background timer killed by OS | High | Save start timestamp + elapsed to SharedPrefs on every tick (via `TimerService._persistState()`) |
| `ReorderableListView` conflicting with `Dismissible` | Medium | Wrap `Dismissible` inside `ReorderableListView` correctly (key must be on `Dismissible`, not card widget); test both gestures separately |
| Drift query performance on large sessions history | Low | Indexes on `started_at` + `card_id` from Sprint 1; write Drift migrations from day 1 |
| Deep Work auto-end feels abrupt to user | Medium | Show `PauseBarIndicator` urgency color change at 30s remaining + vibrate warning at 10s |
| Clay shadows performance (many cards) | Low | Use `RepaintBoundary` on each `StrangeCardWidget`; avoid rebuilding shadow on scroll |
| Dark mode Claymorphism legibility | Medium | Use `Clay Shadow Primary` with higher opacity in dark mode (0.40 vs 0.25 in light) |
| Locale not updating live after language change | Low | Re-assign `MaterialApp.locale` from `SettingsCubit` state; wrap `MaterialApp` in a `BlocBuilder<SettingsCubit>` that rebuilds on locale change |
